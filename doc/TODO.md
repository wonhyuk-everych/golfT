## 개발 관련 정리


### 카카오 로그인

### 카카오 채널 이동
- https://developers.kakao.com/docs/latest/ko/kakaotalk-channel/js#check-relationship-sample


### 인증번호 발송
- 인증번호 발송은 CAFE24 에서 SMS 전송하는 서비스 사용하면 될듯
- 인증번호 발급은 서버에서 작업해서 메시지 보내면 될듯


### 결재
- 결재는 CAFE24 에서 결재하는 서비스 사용하면 될듯


### 구글 지도 경로
- https://www.google.com/maps/dir/37.5245522,126.8330309/37.4587231,127.6991705
- 출발, 도착지 위도 경도만 알면 구글 지도 경로를 표현 할 수 있음
- 그대신 새창을 띄워 보여줘야 한다

### 구글 지도 주변 정보
- 위도 경도를 제공 하면 주변 호텔과 같은 장소를 보여주는 기능
- 구글 API 사용해야 할듯
